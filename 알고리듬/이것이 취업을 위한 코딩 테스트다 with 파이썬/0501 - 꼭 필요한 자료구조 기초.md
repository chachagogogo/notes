## 스택
```python
stack = []  
  
stack.append(1)  
stack.append(2)  
stack.append(3)  
stack.append(4)  
  
print(stack)  # [1, 2, 3, 4]  
  
stack.pop()  # 4 리턴  
stack.pop()  # 3 리턴  
  
print(stack)  # [1, 2]  
print(stack[::-1])  # [2, 1]
```

## 큐
```python
from collections import deque  
  
queue = deque()  
  
queue.append(1)  
queue.append(2)  
queue.append(3)  
queue.append(4)  
  
print(queue)  # deque([1, 2, 3, 4])  
print(list(queue))  # [1, 2, 3, 4]  
  
queue.popleft()  # 1 리턴  
queue.popleft()  # 2 리턴  
  
print(queue)  # deque([3, 4])  
print(queue.reverse())  # deque([4, 3])  
print(list(queue))  # [3, 4]
```

## 재귀 함수
```python
def factorial_iterative(n):  
    if n < 1:  
        return 1  
    return n * factorial_iterative(n - 1)
```
